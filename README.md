# Gandi (gandi)

Gandi is a domain name registrar and web hosting provider. The Gandi v5 Public API exposes domain management, LiveDNS, certificates, email, organization, billing, and hosting capabilities for programmatic use.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/gandi/refs/heads/main/apis.yml)

## APIs

### Gandi Domain API
The Gandi Domain API enables you to register, manage, transfer, and renew domain names registered with Gandi.

- **Human URL:** https://api.gandi.net/docs/domains/
- **Base URL:** `https://api.gandi.net/v5/domain`
- **Documentation:** https://api.gandi.net/docs/domains/
- **OpenAPI:** [openapi/domains-openapi-original.yml](openapi/domains-openapi-original.yml)
- **Tags:** Domains, Domain Registrar

### Gandi LiveDNS API
The Gandi LiveDNS API provides DNS management capabilities for domains managed by Gandi, including DNS records, DNSSEC, zone transfers, and TSIG keys.

- **Human URL:** https://api.gandi.net/docs/livedns/
- **Base URL:** `https://api.gandi.net/v5/livedns`
- **Documentation:** https://api.gandi.net/docs/livedns/
- **OpenAPI:** [openapi/livedns-openapi-original.yml](openapi/livedns-openapi-original.yml)
- **Tags:** DNS, Domains
